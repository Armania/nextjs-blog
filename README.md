# READ ME

## Project Setup

Dowload the content of this application, cd inside that directory, you can run several commands:

## Starts the development server

```sh
npm run dev
```

## Builds the app for production

```sh
npm run build
```

## Runs the built app in production mode

```sh
npm start
```
